# Hangman
Terminal Hangman

To install 
```
$git clone https://github.com/Pushkarang/Hangman.git
$cd Hangman
$make
$sudo make install
```

To play 
```
$Hangman [option]
  option: -animal -city -word
```

To add level 
```
$chmod +x addlevel.sh
$sh ./addlevel.sh [levelname] [path_to_list_of_words]
```

To uninstall
```
$sudo make clean
```
